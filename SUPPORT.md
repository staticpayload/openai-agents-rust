# Support

## Where To Ask What

- bug reports: GitHub Issues
- feature requests: GitHub Issues
- docs problems: GitHub Issues
- security reports: follow [SECURITY.md](SECURITY.md)

## Before Opening An Issue

- confirm the behavior on the latest published version or `main`
- include a minimal reproduction when possible
- note the crate, package version, and Rust version

## What To Expect

This is an open-source project, not a commercial support channel. Response time depends on maintainer availability.
