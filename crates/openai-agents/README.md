# openai-agents-rs

Public facade crate for `openai-agents-rust`.

Install:

```toml
[dependencies]
openai-agents = { package = "openai-agents-rs", version = "0.1.2" }
```

Rust import path:

```rust
use openai_agents::{Agent, Runner};
```

- repository: <https://github.com/scalarian/openai-agents-rust>
- docs: <https://docs.rs/openai-agents-rs>
